# Material organico hemico (SiBCS Cap 14)

Material organico em decomposicao intermediaria: 17-40% de fibras
esfregadas OU indice de von Post H5-H6. Discrimina Organossolos Hemicos
no 3o nivel.

## Usage

``` r
hemico(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

## Value

[`DiagnosticResult`](https://hugomachadorodrigues.github.io/soilKey/reference/DiagnosticResult.md).

## References

Embrapa (2018), SiBCS 5a ed., Cap 14 (Organossolos), pp 224-226.
