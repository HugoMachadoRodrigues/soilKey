# Endocalcic qualifier (cam): calcic horizon between 50 and 100 cm.

WRB 2022 Ch 5 (depth-conditional supplementary form of Calcic).
Referenced in Chernozems Ch 4. The diagnostic is the same as Calcic; the
difference is the depth band – Endocalcic requires the calcic horizon to
start at \>= 50 cm (deep, subsoil) rather than within the upper 50 cm.

## Usage

``` r
qual_endocalcic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

## Value

A
[`DiagnosticResult`](https://hugomachadorodrigues.github.io/soilKey/reference/DiagnosticResult.md).
