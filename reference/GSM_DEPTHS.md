# Default GlobalSoilMap depth intervals (cm)

GSM standard per Arrouays et al. (2014) "GlobalSoilMap: Toward a
fine-resolution global grid of soil properties". Boundaries: 0-5, 5-15,
15-30, 30-60, 60-100, 100-200 cm.

## Usage

``` r
GSM_DEPTHS
```

## Format

An object of class `numeric` of length 7.
