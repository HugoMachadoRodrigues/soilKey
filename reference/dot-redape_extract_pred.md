# Extract the predicted label at a given SiBCS level from a classify_sibcs() result

Extract the predicted label at a given SiBCS level from a
classify_sibcs() result

## Usage

``` r
.redape_extract_pred(res, level)
```

## Details

The returned label is a `ClassificationResult` field for the requested
level (Order / Subordem / Grande Grupo / Subgrupo).
