# Plaggic qualifier (pa): plaggic horizon (sod-amended surface).

v0.9.2.C: thin wrapper around the v0.3.3
[`plaggic`](https://hugomachadorodrigues.github.io/soilKey/reference/plaggic.md)
diagnostic now that the anthropic-evidence gate (P / artefacts /
Apl-family designation) lives inside the diagnostic itself. The v0.9.1
qualifier-side gate is therefore retired.

## Usage

``` r
qual_plaggic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
