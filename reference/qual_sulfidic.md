# Sulfidic qualifier (sf): hyper- OR hyposulfidic material in upper 100 cm (the WRB Sulfidic qualifier covers either acidification class).

Sulfidic qualifier (sf): hyper- OR hyposulfidic material in upper 100 cm
(the WRB Sulfidic qualifier covers either acidification class).

## Usage

``` r
qual_sulfidic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
