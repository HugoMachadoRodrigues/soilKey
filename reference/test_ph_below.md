# Test that ph_h2o is at or below a threshold

Default 5.9 (Spodic horizon supplementary criterion, WRB 2022).

## Usage

``` r
test_ph_below(h, max_ph = 5.9, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- max_ph:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
