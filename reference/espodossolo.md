# Espodossolos (SiBCS Cap 4, p 112; conceito Cap 3, p 90-91)

Horizonte B espodico imediatamente abaixo de horizontes E ou A, dentro
de 200 cm (ou 400 cm se A+E ou histico+E ultrapassam 200).

## Usage

``` r
espodossolo(pedon, max_top_cm = 200)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- max_top_cm:

  Numeric threshold or option (see Details).
