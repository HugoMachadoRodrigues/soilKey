# Carater alitico (SiBCS Cap 1, p 32)

Critérios canônicos: Al(extr) \>= 4 cmolc/kg solo, saturacao por
aluminio \[100 \* Al / (S + Al)\] \>= 50%, e saturacao por bases V \<
50%. Avaliado no horizonte B (ou C, na ausencia de B).

## Usage

``` r
carater_alitico(pedon, min_al = 4, min_al_sat = 50, max_v = 50)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- min_al:

  Numeric threshold or option (see Details).

- min_al_sat:

  Numeric threshold or option (see Details).

- max_v:

  Numeric threshold or option (see Details).
