# Protic qualifier (pr): Arenosol (or Regosol) with NO incipient subsurface horizon – i.e. an A-over-C profile where no cambic, no argic, no spodic, no ferralic, no nitic horizon is present in the upper 100 cm. v0.9.1 implements as the conjunction of the "no B horizon" diagnostics.

Protic qualifier (pr): Arenosol (or Regosol) with NO incipient
subsurface horizon – i.e. an A-over-C profile where no cambic, no argic,
no spodic, no ferralic, no nitic horizon is present in the upper 100 cm.
v0.9.1 implements as the conjunction of the "no B horizon" diagnostics.

## Usage

``` r
qual_protic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
