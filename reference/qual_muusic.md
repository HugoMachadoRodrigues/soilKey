# Muusic qualifier (mu): high-fibre peat (non-moss-specific)

WRB 2022 Ch 5 (Histosols): "Containing \>= 75% by volume rubbed fibres
in organic material \>= 40 cm thick within 100 cm."

## Usage

``` r
qual_muusic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
