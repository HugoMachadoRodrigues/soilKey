# SmartSolos drainage class scale (DRENAGEM, 1-8)

SiBCS / Embrapa drainage scale used by the SmartSolosExpert API: 1
excessivamente drenado .. 8 muito mal drenado. soilKey does not have a
canonical drainage column yet; user supplies via `drenagem` argument
when known.

## Usage

``` r
.SMARTSOLOS_DRAINAGE_SCALE
```

## Format

An object of class `integer` of length 8.
