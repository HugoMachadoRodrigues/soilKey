# Haplic qualifier (ha): no other principal qualifier of the RSG applies. Always passes; the qualifier resolution machinery uses it as the default when no other qualifier matched.

Haplic qualifier (ha): no other principal qualifier of the RSG applies.
Always passes; the qualifier resolution machinery uses it as the default
when no other qualifier matched.

## Usage

``` r
qual_haplic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
