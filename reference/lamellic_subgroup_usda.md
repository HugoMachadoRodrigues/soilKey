# Lamellic Subgroup helper (Spodosols Haplorthods)

Pass when 2+ lamellae (clay-rich bands \< 7.5 cm thick) are present
below the spodic horizon. v0.8: detected via designation containing "&"
(lamella notation in KST) OR multiple thin clay-bumps in clay_pct.

## Usage

``` r
lamellic_subgroup_usda(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

## Value

A
[`DiagnosticResult`](https://hugomachadorodrigues.github.io/soilKey/reference/DiagnosticResult.md).
