# Build the canonical Andosol fixture

Synthetic Andosol on volcanic tephra: very dark surface with low bulk
density (0.7 g/cm^3) and high active Al + Fe (Al_ox + 0.5 \* Fe_ox =
2.25%). By construction
[`andic_properties`](https://hugomachadorodrigues.github.io/soilKey/reference/andic_properties.md)
passes.

## Usage

``` r
make_andosol_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
