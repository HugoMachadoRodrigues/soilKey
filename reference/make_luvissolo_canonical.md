# Perfil canonico de Luvissolo (SiBCS 5a ed., Cap 11)

Solo com B textural argila Ta + V alta. Tipico do semiarido com rocha
basica.

## Usage

``` r
make_luvissolo_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md)
populated with the canonical horizons and site metadata for this
reference profile.
