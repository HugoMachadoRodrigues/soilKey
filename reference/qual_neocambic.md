# Neocambic qualifier (nc): "young" cambic horizon with weak development

WRB 2022 Ch 5 (Retisols): "Cambic horizon with structure_grade \\weak\\
only (early-stage pedogenesis)." Composite: cambic + weak structure.

## Usage

``` r
qual_neocambic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
