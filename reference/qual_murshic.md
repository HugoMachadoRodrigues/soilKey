# Murshic qualifier (mr): partly drained organic with strong decomposition

WRB 2022 Ch 5 (Histosols): "Drained organic soils with sapric
decomposition (rubbed fibres \< 17%) and von Post \>= 7 in upper 50 cm."
Proxy via low rubbed fibre + von Post (when present).

## Usage

``` r
qual_murshic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
