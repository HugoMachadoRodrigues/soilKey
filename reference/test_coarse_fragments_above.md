# Test coarse-fragments percent above a threshold

WRB 2022 alternative criterion for the leptic feature: coarse fragments
\>= `min_pct` % by volume in a layer within `max_depth` of the surface.
Used as an OR-alternative to the R / Cr designation pattern.

## Usage

``` r
test_coarse_fragments_above(h, min_pct = 90, max_top_cm = 25)
```
