# Aric qualifier (ar): mineral surface horizon homogenised by ploughing – designation pattern `Ap`, `Apk`, `Apc`, etc., starting within the upper 30 cm.

Aric qualifier (ar): mineral surface horizon homogenised by ploughing –
designation pattern `Ap`, `Apk`, `Apc`, etc., starting within the upper
30 cm.

## Usage

``` r
qual_aric(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
