# Pellic qualifier (pe): in the upper 30 cm, Munsell value \<= 4 moist AND chroma \<= 2 moist. Diagnostic of "black" (dark) Vertisols.

Pellic qualifier (pe): in the upper 30 cm, Munsell value \<= 4 moist AND
chroma \<= 2 moist. Diagnostic of "black" (dark) Vertisols.

## Usage

``` r
qual_pellic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
