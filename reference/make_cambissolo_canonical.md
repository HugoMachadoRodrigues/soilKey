# Perfil canonico de Cambissolo (SiBCS 5a ed., Cap 6)

Reusa fixture WRB Cambisol – B incipiente sem ser plintico, vertico,
planico, etc.

## Usage

``` r
make_cambissolo_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md)
populated with the canonical horizons and site metadata for this
reference profile.
