# Material organico saprico (SiBCS Cap 14)

Material organico altamente decomposto: \< 17% de fibras esfregadas OU
indice de von Post H7-H10. Discrimina Organossolos Sapricos no 3o nivel
categorico.

## Usage

``` r
saprico(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

## Value

[`DiagnosticResult`](https://hugomachadorodrigues.github.io/soilKey/reference/DiagnosticResult.md).

## References

Embrapa (2018), SiBCS 5a ed., Cap 14 (Organossolos), pp 224-226.
