# Neobrunic qualifier (nb): "young" cambic-like horizon

WRB 2022 Ch 5 (Retisols): "Cambic horizon-like alteration that has
formed in the last few centuries (recent agricultural, colluvial, or
volcanic deposits)." Composite: cambic + recent-age marker via
`layer_origin` matching young-soil patterns.

## Usage

``` r
qual_neobrunic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
