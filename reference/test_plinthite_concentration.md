# Test for plinthite concentration above threshold (per layer)

Default 15% by volume (plinthic horizon, WRB 2022 Chapter 3). Used by
[`plinthic`](https://hugomachadorodrigues.github.io/soilKey/reference/plinthic.md).

## Usage

``` r
test_plinthite_concentration(h, min_pct = 15, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- min_pct:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
