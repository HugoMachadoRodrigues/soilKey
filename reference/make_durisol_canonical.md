# Build the canonical Durisol fixture

Synthetic semi-arid Durisol with a Si-cemented subsurface horizon (35%
duripan nodules over 45 cm). By construction
[`duric_horizon`](https://hugomachadorodrigues.github.io/soilKey/reference/duric_horizon.md)
passes on Bdu.

## Usage

``` r
make_durisol_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
