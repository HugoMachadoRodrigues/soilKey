# Horizonte concrecionario (SiBCS Cap 2, p 68-69; v0.7)

\\= 50% volume material grosso (predominio petroplintita) numa matriz,
espessura \\= 30 cm. Designation Ac/Ec/Bc/Cc.

## Usage

``` r
horizonte_concrecionario(
  pedon,
  min_petroplinthite_pct = 50,
  min_thickness = 30
)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- min_petroplinthite_pct:

  Numeric threshold or option (see Details).

- min_thickness:

  Numeric threshold or option (see Details).
