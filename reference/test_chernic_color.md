# Test for chroma \<= 2 (moist) within the upper part of the profile

Default upper boundary is 20 cm (Chernozem criterion: dark colour in the
upper 20 cm of the mollic horizon).

## Usage

``` r
test_chernic_color(h, max_top_cm = 20, max_chroma = 2, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- max_top_cm:

  Numeric threshold or option (see Details).

- max_chroma:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
