# Neossolos (SiBCS Cap 4, p 111-112; conceito Cap 3, p 96-97)

Solos pouco evoluidos: SEM horizonte B diagnostico + ausencia de: (a)
glei dentro 150 cm, (b) plintico dentro 40 cm, (c) vertico imediatamente
abaixo de A, (d) A chernozemico conjugado com carbonatico ou cálcico.

## Usage

``` r
neossolo(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
