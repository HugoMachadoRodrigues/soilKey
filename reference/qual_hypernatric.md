# Hypernatric supplementary qualifier (hyna): very high Na (\>= 70% ESP)

## Usage

``` r
qual_hypernatric(pedon)
```

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
