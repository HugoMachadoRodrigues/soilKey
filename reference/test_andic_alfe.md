# Test the andic Al/Fe oxalate criterion: (al_ox + 0.5\*fe_ox) \>= 2.0%

Distinct from spodic (which uses 0.5%); the andic threshold is four
times higher per WRB 2022 Chapter 3.

## Usage

``` r
test_andic_alfe(h, min_pct = 2, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- min_pct:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
