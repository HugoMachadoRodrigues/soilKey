# Bryic supplementary qualifier (by): bryophyte cover at surface

WRB 2022 Ch 5: "Predominant bryophyte (moss / liverwort) ground cover."
Implementation: `layer_origin` matches moss / lichen pattern OR
`vegetation_cover` site field \>= 50.

## Usage

``` r
qual_bryic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
