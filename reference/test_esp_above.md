# Test exchangeable sodium percentage above threshold

Default 15% (natric horizon, WRB 2022 Chapter 3).

## Usage

``` r
test_esp_above(h, min_pct = 15, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- min_pct:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
