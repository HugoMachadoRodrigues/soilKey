# Test that a character column matches a regex per layer

Test that a character column matches a regex per layer

## Usage

``` r
test_pattern_match(h, column, pattern, candidate_layers = NULL)
```

## Arguments

- h:

  Horizons table.

- column:

  Character column name.

- pattern:

  Regex (case-insensitive).

- candidate_layers:

  Optional restriction.

## Value

Sub-test result.
