# Test that artefacts_pct \>= threshold within the upper max_top_cm

Default 20% by volume (Technosols criterion, WRB 2022).

## Usage

``` r
test_artefacts_concentration(
  h,
  min_pct = 20,
  max_top_cm = 100,
  candidate_layers = NULL
)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- min_pct:

  Numeric threshold or option (see Details).

- max_top_cm:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
