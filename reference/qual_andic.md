# Andic qualifier (an): andic OR vitric properties combined \>= 30 cm. v0.9 simplification: passes if andic_properties or vitric_properties passes within 100 cm.

Andic qualifier (an): andic OR vitric properties combined \>= 30 cm.
v0.9 simplification: passes if andic_properties or vitric_properties
passes within 100 cm.

## Usage

``` r
qual_andic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
