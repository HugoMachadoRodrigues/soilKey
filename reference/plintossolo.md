# Plintossolos (SiBCS Cap 4, p 113; conceito Cap 3, p 102-104)

Horizonte plintico (nao coincidente com B planico de carater sodico), OR
litoplintico, OR concrecionario, iniciando dentro de 40 cm OR dentro de
200 cm precedido de glei OR A/E OR horizonte com cores palidas /
variegadas / mosqueados.

## Usage

``` r
plintossolo(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
