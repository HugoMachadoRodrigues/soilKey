# Archaic supplementary qualifier (ah): archeological context

WRB 2022 Ch 5: "Soil developed in or affected by ancient cultural
material (\>1000 yr old)." Detects via `contamination_type` matching
"archaeological" or site-level cultural-period field.

## Usage

``` r
qual_archaic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
