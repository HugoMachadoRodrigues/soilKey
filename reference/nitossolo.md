# Nitossolos (SiBCS Cap 4, p 114; conceito Cap 3, p 97-98)

\\= 350 g/kg argila incluindo no horizonte A, com B nitico abaixo do A,
com argila ativ baixa OR ativ alta + carater alumínico, na maior parte
dos primeiros 100 cm do B (incl. BA).

## Usage

``` r
nitossolo(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
