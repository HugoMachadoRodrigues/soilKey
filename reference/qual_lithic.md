# Lithic qualifier (lt): continuous rock starting within 10 cm. Tighter depth gate than Leptic (which is \<= 100 cm) and Nudilithic (== 0 cm).

Lithic qualifier (lt): continuous rock starting within 10 cm. Tighter
depth gate than Leptic (which is \<= 100 cm) and Nudilithic (== 0 cm).

## Usage

``` r
qual_lithic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
