# Test for high free-iron content (`fe_dcb_pct` \>= threshold)

Default 4% (an indicator of strong red colour and Fe-richness; used as a
v0.3 simplified marker for nitic horizon's typical Fe content).

## Usage

``` r
test_fe_dcb_above(h, min_pct = 4, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- min_pct:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
