# Test that base saturation is below a threshold

Default 50% (Acrisol RSG criterion). Reads `bs_pct`.

## Usage

``` r
test_bs_below(h, max_pct = 50, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- max_pct:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
