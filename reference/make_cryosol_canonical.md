# Build the canonical Cryosol fixture

Synthetic Arctic Cryosol on weathered shale with permafrost at 50 cm:
thawed A horizon over a frozen Bf horizon. By construction
[`cryic_conditions`](https://hugomachadorodrigues.github.io/soilKey/reference/cryic_conditions.md)
passes via the designation pattern.

## Usage

``` r
make_cryosol_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
