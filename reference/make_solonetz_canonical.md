# Build the canonical Solonetz fixture

Synthetic Solonetz on saline-sodic substrate: argic Btn with columnar
structure and high exchangeable Na (ESP ~ 28%). By construction
[`natric_horizon`](https://hugomachadorodrigues.github.io/soilKey/reference/natric_horizon.md)
passes.

## Usage

``` r
make_solonetz_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
