# Profundihumic qualifier (ph): SOC \>= 1.4% to depth \>= 100 cm

WRB 2022 Ch 5 (Nitisols / Ferralsols): "Containing \>= 1.4% organic
carbon (by weight, excluding live fine roots) as a weighted average from
the soil surface down to 100 cm."

## Usage

``` r
qual_profundihumic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
