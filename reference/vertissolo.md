# Vertissolos (SiBCS Cap 4, p 112; conceito Cap 3, p 105-106)

superficiais + fendas verticais + ausencia de contato litico /
petrocalcico / duripa nos 30 cm + COLE \\= 0.06.

## Usage

``` r
vertissolo(pedon)
```
