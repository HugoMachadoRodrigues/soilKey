# Test that organic carbon is at or above a threshold

Default 12% (histic horizon, WRB 2022 Chapter 3).

## Usage

``` r
test_oc_above(h, min_pct = 12, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- min_pct:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
