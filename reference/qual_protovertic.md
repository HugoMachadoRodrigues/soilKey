# Protovertic qualifier (qv): protovertic horizon (vertic-spectrum lower bound, no slickensides yet but the clay + structure / shrink-swell signal is already present) within the upper 100 cm. Wraps [`protovertic`](https://hugomachadorodrigues.github.io/soilKey/reference/protovertic.md) and is mutually exclusive with the strict Vertic qualifier.
