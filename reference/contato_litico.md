# Contato litico (SiBCS Cap 1, p 40): rocha continua dura. Reuso de [`continuous_rock`](https://hugomachadorodrigues.github.io/soilKey/reference/continuous_rock.md) via designacao R / Cr.

Contato litico (SiBCS Cap 1, p 40): rocha continua dura. Reuso de
[`continuous_rock`](https://hugomachadorodrigues.github.io/soilKey/reference/continuous_rock.md)
via designacao R / Cr.

## Usage

``` r
contato_litico(pedon, max_depth_cm = NULL)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- max_depth_cm:

  Profundidade maxima do contato (default `NULL`). SiBCS Cap 14
  Subgrupos liticos de Folicos usam `max_depth_cm = 50`.
