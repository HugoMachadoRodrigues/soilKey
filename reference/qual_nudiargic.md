# Nudiargic qualifier (nu): argic horizon at the surface

WRB 2022 Ch 5 (Acrisols / Lixisols / Alisols / Luvisols / Retisols):
"Argic horizon starting \<= 5 cm from the soil surface (no overlying
eluvial / albic / mollic / umbric layer)."

## Usage

``` r
qual_nudiargic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
