# Horizonte litoplintico (SiBCS Cap 2, p 69; v0.7)

Petroplintita continua (ironstone). Reuso de
[`petroplinthic`](https://hugomachadorodrigues.github.io/soilKey/reference/petroplinthic.md)
(WRB), espessura \\= 10 cm.

## Usage

``` r
horizonte_litoplintico(pedon, min_thickness = 10)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- min_thickness:

  Numeric threshold or option (see Details).
