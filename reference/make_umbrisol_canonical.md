# Build the canonical Umbrisol fixture

Synthetic humid-temperate Umbrisol on weathered acidic schist: deep
organic-rich dark surface with low base saturation – the acid analogue
of a Phaeozem. By construction
[`umbric_horizon`](https://hugomachadorodrigues.github.io/soilKey/reference/umbric_horizon.md)
passes;
[`mollic`](https://hugomachadorodrigues.github.io/soilKey/reference/mollic.md)
fails on BS \< 50.

## Usage

``` r
make_umbrisol_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
