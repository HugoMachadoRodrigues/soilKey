# Carater palico (SiBCS Cap 11)

Solos com espessura do solum (A + B, inclusive E e exclusive BC) maior
que `min_solum_cm` (default 80). Discrimina os Grandes Grupos Palicos de
Luvissolos (Cap 11 TCp, TXp) – "desenvolvimento excessivo" (do latim
"pale").

## Usage

``` r
carater_palico(pedon, min_solum_cm = 80)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- min_solum_cm:

  Default 80.

## Value

[`DiagnosticResult`](https://hugomachadorodrigues.github.io/soilKey/reference/DiagnosticResult.md).

## References

Embrapa (2018), SiBCS 5a ed., Cap 11, p 214.
