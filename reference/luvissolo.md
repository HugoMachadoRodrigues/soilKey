# Luvissolos (SiBCS Cap 4, p 113; conceito Cap 3, p 95-96)

Horizonte B textural com argila ativ alta E saturacao por bases alta (V
\\= 50%) na maior parte dos primeiros 100 cm do B (incl. BA), abaixo de
A ou E.

## Usage

``` r
luvissolo(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
