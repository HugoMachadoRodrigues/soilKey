# Test that clay_pct is at or above a threshold

Default 30% (vertic features minimum, WRB 2022 Chapter 3).

## Usage

``` r
test_clay_above(h, min_pct = 30, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- min_pct:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
