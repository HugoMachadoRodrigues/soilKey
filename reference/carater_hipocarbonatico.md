# Carater hipocarbonatico (SiBCS Cap 1, p 33): CaCO3 entre 50 e 150 g/kg.

Carater hipocarbonatico (SiBCS Cap 1, p 33): CaCO3 entre 50 e 150 g/kg.

## Usage

``` r
carater_hipocarbonatico(pedon, max_depth_cm = NULL)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- max_depth_cm:

  Profundidade maxima em que camadas qualificam (default `NULL` = sem
  restricao). SiBCS Cap 14 Subgrupos de Organossolos Haplicos Sapricos
  usam `max_depth_cm = 150`.
