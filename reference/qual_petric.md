# Petric qualifier (pt): any petro-cemented horizon (petrocalcic / petroduric / petrogypsic / petroplinthic) within 100 cm.

Petric qualifier (pt): any petro-cemented horizon (petrocalcic /
petroduric / petrogypsic / petroplinthic) within 100 cm.

## Usage

``` r
qual_petric(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
