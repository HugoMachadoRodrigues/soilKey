# Build the canonical Technosol fixture

Synthetic urban / industrial Technosol: surface horizon with 30%
anthropogenic artefacts (brick, glass, slag, plastic). By construction
[`technic_features`](https://hugomachadorodrigues.github.io/soilKey/reference/technic_features.md)
passes.

## Usage

``` r
make_technosol_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
