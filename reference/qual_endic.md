# Endic supplementary qualifier (ec): generic "in deep horizon" marker

WRB 2022 Ch 5: generic "Endo-X" prefix marker for any qualifier that
takes a depth window 50-100 cm. Without a base diagnostic it returns NA;
in practice it is composed with another qualifier.

## Usage

``` r
qual_endic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
