# Compose the canonical reference label for a given SiBCS level

Concatenates the relevant Redape reference fields (singular Portuguese
nominal phrase) and applies plural-canonical normalisation so the result
is comparable to the soilKey predicted label (e.g.\\ "Argissolos
Amarelos Distroficos abrupticos").

## Usage

``` r
.redape_compose_ref(pedon, level)
```
