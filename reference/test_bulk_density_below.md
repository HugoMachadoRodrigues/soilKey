# Test that bulk density is at or below a threshold

Default 0.9 g/cm^3 (andic property, WRB 2022).

## Usage

``` r
test_bulk_density_below(h, max_g_cm3 = 0.9, candidate_layers = NULL)
```

## Arguments

- h:

  Numeric threshold or option (see Details).

- max_g_cm3:

  Numeric threshold or option (see Details).

- candidate_layers:

  Numeric threshold or option (see Details).
