# Isolatic qualifier (il): isolated technic material

WRB 2022 Ch 5 (Technosols): "Containing isolated bodies of technic hard
material (concrete blocks, asphalt slabs, brick walls) but NOT covering
the full surface." Detection via `artefacts_urbic_pct` or
`artefacts_industrial_pct` between 5 and 50%.

## Usage

``` r
qual_isolatic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
