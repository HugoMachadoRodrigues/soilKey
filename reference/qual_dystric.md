# Dystric qualifier (dy), WRB 2022 Ch 5.

Exchangeable Al \\\>\\ exchangeable bases (Ca+Mg+K+Na) in half or more
of the combined thickness of mineral layers between 20 and 100 cm
(organic layers use the Histosol pH_water \< 5.5 branch). Uses
`al_sat_pct` or `al_cmol` vs the base cations; no base-saturation
fallback (strict).

## Usage

``` r
qual_dystric(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
