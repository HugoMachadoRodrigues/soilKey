# Horizonte calcico (SiBCS Cap 2, p 71-72; v0.7)

Reuso direto de
[`calcic`](https://hugomachadorodrigues.github.io/soilKey/reference/calcic.md)
(WRB Ch 3.1.5) – criterios identicos: 150 g/kg CaCO3 + 50 g/kg a mais
que sub-jacente + espessura \\= 15 cm.

## Usage

``` r
horizonte_calcico(pedon, ...)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).

- ...:

  Reserved for future arguments.
