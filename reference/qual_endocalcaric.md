# Endocalcaric qualifier (cae): calcaric only at depth \>= 50 cm

WRB 2022 Ch 5 (Umbrisols / Retisols): "Calcaric material starting \>= 50
cm from the soil surface." Modifier of
[`calcaric_material`](https://hugomachadorodrigues.github.io/soilKey/reference/calcaric_material.md).

## Usage

``` r
qual_endocalcaric(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
