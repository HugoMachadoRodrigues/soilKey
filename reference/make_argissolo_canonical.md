# Perfil canonico de Argissolo (SiBCS 5a ed., Cap 5)

B textural com gradiente significativo, argila ativ baixa ou alta + V
baixa. Catch-all final na chave – tipica do Brasil tropical.

## Usage

``` r
make_argissolo_canonical()
```

## Value

A
[`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md)
populated with the canonical horizons and site metadata for this
reference profile.
