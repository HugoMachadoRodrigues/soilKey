# Anthromollic qualifier (am): anthric horizon overlying spodic

WRB 2022 Ch 5 (Podzols): "Having an anthric (irrigation / Plaggic-like)
surface horizon directly over spodic / albic / diagnostic horizon."
Combines
[`anthric_horizons`](https://hugomachadorodrigues.github.io/soilKey/reference/anthric_horizons.md) +
overlying-spodic check.

## Usage

``` r
qual_anthromollic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
