Cutanic qualifier (ct): visible illuvial clay coatings on argic- horizon
ped surfaces (the "Cutanic Luvisol" / "Cutanic Argissol" signature).
v0.9.1: argic horizon passes AND the schema column `clay_films_amount`
contains "common", "many", or "continuous" (or "shiny" – common
Brazilian descriptor for nitic surfaces) in some argic layer.

## Usage

``` r
qual_cutanic(pedon)
```

## Arguments

- pedon:

  A
  [`PedonRecord`](https://hugomachadorodrigues.github.io/soilKey/reference/PedonRecord.md).
